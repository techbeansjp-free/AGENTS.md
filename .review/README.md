# レビューディレクトリ

このディレクトリには、**AGENTS 規約とテンプレート全体のレビュー結果**を時系列で管理します。

> **重要**: このディレクトリのレビューは、**issue/タスクごとのレビュー（`04_review.md`）とは異なります**。
>
> - **`.review/`**: AGENTS 規約とテンプレート全体のレビュー（このディレクトリ）
> - **`04_review.md`**: 各 issue/タスクの実装完了後のレビュー（`.workflow/{YYYYMMDD_issue_name}/04_review.md`）

---

## 目的

このディレクトリは、以下の目的で使用します：

1. **AGENTS 規約のレビュー**: 規約ドキュメント（`AGENTS.md`、`AGENTS_AI_PLAYBOOK.md`）の整合性や改善点の記録
2. **テンプレートのレビュー**: テンプレートファイル（`.workflow/templates/`）の整合性や改善点の記録
3. **ワークフローのレビュー**: ワークフロー全体の有効性や改善点の記録
4. **AI エージェントの動作確認**: AI エージェントが規約に従って正しく動作するかの確認

---

## ファイル命名規則

レビューファイルは以下の形式で命名します：

```
YYYYMMDD_レビュー内容.md
```

**命名例**:

- `20251123_AGENTS規約とテンプレートレビュー.md`
- `20251201_テンプレート更新レビュー.md`
- `20251215_ワークフロー改善レビュー.md`

---

## AI エージェント向けの注意事項

> **⚠️ 重要**: LLM エージェント（AI）は、このディレクトリのレビューファイルを**参照のみ**として使用してください。

### AI エージェントが守るべきこと

1. **参照のみ**: `.review/`ディレクトリのレビューファイルは、過去のレビュー結果を参照するために使用します。新しいレビューを実施する場合は、人間が作成してください。

2. **`04_review.md`との区別**:

   - `.review/`のレビュー: AGENTS 規約とテンプレート全体のレビュー（このディレクトリ）
   - `04_review.md`: 各 issue/タスクの実装完了後のレビュー（`.workflow/{YYYYMMDD_issue_name}/04_review.md`）

3. **ワークフローでの使用**: AI エージェントは、通常のワークフロー（00 → 01 → 02 → 03 → 実装 → 04_review）で `04_review.md` を作成・更新します。`.review/`ディレクトリのレビューファイルは、規約やテンプレートの改善時に参照するためのものです。

### AI エージェントが参照すべきファイル

- **通常のワークフロー**: `.workflow/{YYYYMMDD_issue_name}/04_review.md` を作成・更新
- **規約・テンプレートの改善時**: `.review/`ディレクトリのレビューファイルを参照（過去の改善点を確認）

## レビューの実施タイミング

以下のような場合にレビューを実施します：

1. **規約の更新時**: `AGENTS.md`や`AGENTS_AI_PLAYBOOK.md`を更新した際
2. **テンプレートの更新時**: `.workflow/templates/`のテンプレートファイルを更新した際
3. **ワークフローの改善時**: ワークフロー全体を見直す際
4. **定期的な見直し**: 規約やテンプレートの有効性を確認する際
